# GMP 근무표 자동 생성기

기존 엑셀 근무표를 붙여넣어 시작하고, 월 단위로 검증/통계/자동배정을 지원하는 Windows용 데스크톱 앱입니다.

## 기본 사용 방식

사용자는 기존 엑셀 근무표를 먼저 붙여넣습니다.

표 형식:

| 성명 | 사번 | 1 | 2 | 3 | 4 | ... | 말일 |
|---|---|---|---|---|---|---|---|
| 홍길동 | 1001 | D | S | G/지근 | 지휴 | ... |  |
| 김철수 | 1002 | S | D | 당직 | 지휴 | ... | D |

앱에서 `직원 관리` 탭의 붙여넣기 박스에 표를 통째로 붙여넣고 `기존 근무표 붙여넣기 반영`을 누르면 됩니다.

샘플:

```text
samples/schedule_table.tsv
```

## 근무 코드

- `D` = Day
- `S` = Swing
- `G/지근` = 평일 GY 첫날 표시
- `당직` = 주말/휴일 GY
- `지휴` = GY 이후 휴식, 근무 카운트 제외
- 빈칸 = 휴무

## 주요 기능

- 기존 엑셀 근무표 전체 붙여넣기 import
- 월별 근무표 자동 생성
- 평일 `D / S / G·지근` 공평 분배
- 휴일·주말 `D / S / 당직` 공평 분배
- 대한민국 공휴일 자동 반영 (`holidays` 패키지 사용)
- `G/지근` 또는 `당직` 다음날 `지휴` 자동 확보
- 개인 불가일 배정 제외
- 신규 인원 교육 목적 순환 배치
  - 평일 D 또는 S
  - 휴일 D 또는 S
  - 평일 G/지근
- 엑셀처럼 셀 직접 수정 / 붙여넣기 가능
- 실시간 검증
  - 최소 인원 부족
  - 불가일 배정 위반
  - 당직 평일 배정 경고
  - 휴일 GY 표기 경고
  - 연속 근무 초과
  - 연속 GY 초과
- 사람별 통계와 평균 대비 편차 표시
- 엑셀 저장: 근무표 / 통계 / 검증 시트 포함

## 누적 DB와 근무불가일

### 누적 DB

붙여넣은 월 근무표를 확인한 뒤 툴바의 `현재 근무표 DB 저장`을 누르면 로컬 SQLite DB에 저장됩니다.

DB 파일:

```text
gmp_scheduler.sqlite3
```

저장된 모든 월 기준 누적 통계는 `통계` 탭의 `누적 통계 새로고침`으로 확인합니다.

### 회색 셀 근무불가일

근무불가일은 셀 글자가 아니라 엑셀 배경색으로 받는 방식이므로, 붙여넣기가 아니라 실제 `.xlsx` 파일을 읽어야 합니다.

사용 방법:

1. 직원이 회색으로 칠한 근무불가일 엑셀 파일 준비
2. 앱에서 연도/월 선택
3. 기존 근무표 또는 직원 목록을 먼저 반영
4. 툴바의 `회색 불가일 엑셀` 클릭
5. `.xlsx` 선택

지원 표 형식은 기존 근무표와 동일합니다.

| 성명 | 사번 | 1 | 2 | 3 | ... | 말일 |
|---|---|---|---|---|---|---|
| 홍길동 | 1001 | 회색 |  | 회색 | ... |  |

회색으로 칠한 날짜는 해당 직원의 근무불가일로 저장/검증됩니다.

## Windows에서 BAT로 실행

`run_gmp.bat`을 더블클릭하면 됩니다.

동작:

1. Python 설치 여부 확인
2. Python이 없으면 자동 설치 시도
   - 1순위: `winget`
   - 2순위: python.org 설치 파일 다운로드
3. `.venv` 생성
4. 필요한 패키지 설치
5. 앱 실행

첫 실행은 인터넷 연결이 필요합니다.

## Python으로 직접 실행

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

## Windows EXE

Python 없이 바로 실행하려면 GitHub Releases에서 `GMP-Scheduler.exe`를 받으면 됩니다.

```text
https://github.com/bhmin2100-stack/gmp/releases/tag/windows-latest
```

## 자동 배정 기준

1. 월 단위 공평성 우선
2. 평일/휴일 근무 유형별 횟수 별도 균형
3. 개인 불가일 배정 금지
4. 연속 근무와 연속 GY 최소화
5. 신규 인원 필수 경험 근무 우선 배정
