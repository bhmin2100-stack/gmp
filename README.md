# GMP 근무표 자동 생성기

기존 엑셀 근무표를 붙여넣어 시작하고, 월 단위로 검증/통계/자동배정을 지원하는 Windows용 데스크톱 앱입니다.

## 기본 사용 방식

사용자는 기존 엑셀 근무표를 먼저 붙여넣습니다.

표 형식:

| 성명 | 사번 | 1 | 2 | 3 | 4 | ... | 말일 |
|---|---|---|---|---|---|---|---|
| 홍길동 | 1001 | D | S | G/지근 | 지휴 | ... |  |
| 김철수 | 1002 | S | D | 당직 | 지휴 | ... | D |

앱 메인 화면에서 `월간 근무표` 탭을 보고, 엑셀에서 표 범위를 복사한 뒤 `엑셀 근무표 붙여넣기` 버튼을 누르면 됩니다. 텍스트 박스에 붙여넣는 방식이 아니라 클립보드의 표 정보를 직접 읽습니다.

샘플:

```text
samples/schedule_table.tsv
```

## 메인 화면

앱 첫 탭은 `연간 보기`입니다.

- 선택한 연도의 1월~12월 근무표를 세로로 모두 보여줍니다.
- 마우스 휠로 위아래로 스크롤하면서 달력처럼 확인합니다.
- 각 월 표를 클릭하고 `Ctrl+V` 하면 그 월에 엑셀 근무표가 바로 붙습니다.
- 빈 월도 `여기에 클릭 후 Ctrl+V` 행이 있어 바로 붙여넣을 수 있습니다.
- DB에 저장된 월은 자동으로 채워집니다.
- 현재 편집 중인 월은 `현재 편집 중`으로 표시됩니다.
- 한 달을 크게 편집하려면 `월간 근무표` 탭을 사용합니다.

## V11/V12 기준일 분할

팀 분할 기준일이 정해지면 상단에서 `V11/V12 분할 사용`을 켜고 기준일을 선택합니다.

- `보기`에서 `기존`, `V11`, `V12` 저장본을 전환합니다.
- 기준일 이전 월은 기존 근무표만 열립니다.
- 기준일이 들어 있는 월에서 V11/V12를 보면 기준일 이전 날짜는 기존 근무표로 표시되고 편집되지 않습니다.
- 기준일 이후 날짜는 V11/V12가 각각 별도 저장본으로 DB에 저장됩니다.
- V11/V12 저장본이 아직 없으면 기존 월 근무표를 복사한 초안으로 시작하므로, 필요한 인원/근무만 수정해서 저장하면 됩니다.

## 근무 코드

- `D` = Day
- `S` = Swing
- `G/지근` = 평일 GY 첫날 표시
- `당직` = 주말/휴일 GY
- `지휴` = GY 이후 휴식, 근무 카운트 제외
- 빈칸 = 휴무

## 주요 기능

- 기존 엑셀 근무표 전체 붙여넣기 import
- 월별 근무표 자동 생성
- 평일 `D / S / G·지근` 공평 분배
- 휴일·주말 `D / S / 당직` 공평 분배
- 대한민국 공휴일 자동 반영 (`holidays` 패키지 사용)
- `G/지근` 또는 `당직` 다음날 `지휴` 자동 확보
- 개인 불가일 배정 제외
- 신규 인원 교육 목적 순환 배치
  - 평일 D 또는 S
  - 휴일 D 또는 S
  - 평일 G/지근
- 엑셀처럼 셀 직접 수정 / 붙여넣기 가능
- 실시간 검증
  - 최소 인원 부족
  - 불가일 배정 위반
  - 당직 평일 배정 경고
  - 휴일 GY 표기 경고
  - 연속 근무 초과
  - 연속 GY 초과
- 사람별 통계와 평균 대비 편차 표시
- 엑셀 저장: 근무표 / 통계 / 검증 시트 포함

## 누적 DB와 근무불가일

### 누적 DB

붙여넣은 월 근무표를 확인한 뒤 툴바의 `현재 근무표 DB 저장`을 누르면 로컬 SQLite DB에 저장됩니다.

DB 파일:

```text
gmp_scheduler.sqlite3
```

저장된 모든 월 기준 누적 통계는 `통계` 탭의 `누적 통계 새로고침`으로 확인합니다.

### 회색 셀 근무불가일

근무불가일은 파일 선택이 아니라 복붙으로 넣습니다. 셀 글자가 아니라 엑셀 배경색으로 받는 방식이므로, 엑셀에서 표 범위를 복사하면 클립보드에 같이 들어오는 HTML 표 스타일을 앱이 읽어 회색 배경 셀을 인식합니다.

사용 방법:

1. 직원이 회색으로 칠한 근무불가일 엑셀 표 범위 선택
2. 복사
3. 앱에서 연도/월 선택
4. 기존 근무표를 먼저 반영
5. `회색 불가일 붙여넣기` 클릭

지원 표 형식은 기존 근무표와 동일합니다.

| 성명 | 사번 | 1 | 2 | 3 | ... | 말일 |
|---|---|---|---|---|---|---|
| 홍길동 | 1001 | 회색 |  | 회색 | ... |  |

회색으로 칠한 날짜는 해당 직원의 근무불가일로 반영되고 검증됩니다. 만약 색상 인식이 안 되면 엑셀에서 범위를 복사한 직후 바로 버튼을 누르세요.

## Windows에서 BAT로 실행

`run_gmp.bat`을 더블클릭하면 됩니다.

동작:

1. Python 설치 여부 확인
2. Python이 없으면 자동 설치 시도
   - 1순위: `winget`
   - 2순위: python.org 설치 파일 다운로드
3. `.venv` 생성
4. 필요한 패키지 설치
5. 앱 실행

첫 실행은 인터넷 연결이 필요합니다.

## Python으로 직접 실행

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

## Windows EXE

Python 없이 바로 실행하려면 GitHub Releases에서 `GMP-Scheduler.exe`를 받으면 됩니다.

```text
https://github.com/bhmin2100-stack/gmp/releases/tag/windows-latest
```

## 자동 배정 기준

1. 월 단위 공평성 우선
2. 평일/휴일 근무 유형별 횟수 별도 균형
3. 개인 불가일 배정 금지
4. 연속 근무와 연속 GY 최소화
5. 신규 인원 필수 경험 근무 우선 배정
